Yoja Main Window

Once logged in, you are presented with the Yoja chat window

UI Features. Initial Indexing Progress

When you first login to Yoja, it will start the process of reading your files and generating embeddings for them. This process may take many hours, depending of the number of documents present in your storage. Indexing progress is indicated by the indexing progress bar, hightlighted in Green Callout 1 below

Here are the other UI features called out below:

  • 1. Indexing Progress Bar: if you hover over the progress bar, the progress percentage is displayed
  • 2. In the left panel bottom, you have the user's name listed. Hovering over the username pops up the settings gear icon. You can configure Dropbox, Clear Chat History, etc.
  • 3. Yoja includes the ability to search chat history. You can also restart a previous chat with new questions
  • 4. In the chat response, the context files chosen by Yoja is listed; clicking on the context source takes you directly to view the document
